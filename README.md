# Sistema de Leilões

Este é um projeto de um sistema para uma casa de leilões, utilizando Java e MySQL.

## Tecnologias utilizadas
- Java
- MySQL
- JDBC
- Git & GitHub

## Como executar
1. Importe o arquivo .sql para o MySQL.
2. Configure a conexão no projeto (usuário, senha, porta, banco).
3. Execute o projeto pela IDE ou pelo terminal conforme instruções.
